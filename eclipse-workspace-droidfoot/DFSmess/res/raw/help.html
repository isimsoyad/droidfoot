Smess

Smess is a chess variant played on a 7x8-board. There are three types of figure: Ninnies, numskulls and brains. The object is to capture the Brain.

Smess-Infos: http://www.chessvariants.com/other.dir/smess.html

Smess-Rules: http://www.hasbro.com/common/instruct/Smess.pdf

You can start a game by choosing the players. Drag a blue-brain-figure onto the blue marked space (its starting space). Drag a red-brain-figure onto the red marked space (its starting space).

H = human player
1 = computer player (bad)
2 = computer player (better)
3 = computer player (good)
4 = computer player (best)

Menu-Items

  • Start or resume / pause
  • Reset (= new game)
  • Help
  • Settings (scaling, screen orientation)
